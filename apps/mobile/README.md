# Mobile App

Expo + React Native shell for the MedStudy OS mobile companion app.

The source entry now lives under `src/`, with placeholder folders for `screens`, `navigation`, `hooks`, `services`, and `state`.

Allowed internal workspace dependencies:

- `@medstudy/contracts`
- `@medstudy/ui-kit`

This app must never import `@medstudy/domain` directly.

No study session flows, checkpoints, or contract logic are implemented during this foundation phase.
